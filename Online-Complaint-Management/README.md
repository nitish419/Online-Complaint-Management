# Online Complaint Management System (Java Swing)

## 📌 Project Overview
A Java-based desktop application utilizing Object-Oriented Programming and Swing GUI to streamline complaint registration and resolution workflows.

## 🛠 Tech Stack
- **Language:** Core Java
- **UI Framework:** Java Swing / AWT
- **Database:** Local File Serialization (.dat files)

## 🚀 Features
- **Role-Based Access:** Distinct User and Admin dashboards.
- **Automated Workflows:** Auto-generated Tracking IDs.
- **State Management:** Complaint tracking from OPEN to RESOLVED.
- **Data Persistence:** Object serialization ensures no data is lost between sessions.

## ⚙️ How to Run
1. Clone this repository.
2. Open in your favorite IDE (IntelliJ/Eclipse).
3. Run `Main.java`.
4. Default Admin Login -> Username: `admin` | Password: `admin123`
